Sobre o Projeto

Conheça mais sobre nossa jornada, tecnologias utilizadas e o que planejamos para o futuro.

Ideia & Objetivo

O projeto foi desenvolvido como uma aplicação web, desenhada para otimizar e centralizar a operação e gestão interna da ONG Turma do Bem. Nosso projeto atua como um hub para a coordenação de voluntários e pacientes, garantindo que o fluxo de trabalho seja eficiente e centralizado em uma só plataforma.

Tecnologias Utilizadas

  • HTML5 - Estrutura semântica moderna para melhor acessibilidade e SEO, utilizando as tags mais recentes e práticas recomendadas pelo W3C.
  • CSS3 - Estilização avançada com variáveis CSS, Grid Layout, Flexbox e animações suaves para criar uma experiência visual impressionante.

Nota: Todas as tecnologias usadas hoje fazem parte do protótipo, novas técnologias serão incluidas conforme o desenvolvimento do projeto.

Roadmap do Projeto

Nosso projeto segue um roadmap em fases, buscando manter a organização para um projeto limpo.

  1. Fase 1 - Estrutura

    Foco em estabelecer a estrutura fundamental do projeto e sua identidade visual, a arquitetura dos diferentes módulos e o planejamento de como fazer a integração dos mesmos.

  2. Fase 2 - Criação dos módulos

    Concentração no desenvolvimento dos módulos centrais responsáveis pelas features da aplicação. Isso envolve a estrutura de cadastro (Pacientes, Dentistas, Funcionários da ONG), o esqueleto do Módulo de Relatórios, Centro de Comunicações e também na criação da estrutura lógica e da interface de usuário do Módulo de Recomendação (Match), definindo os parâmetros iniciais (prioridade, disponibilidade e região) que serão usados para sugerir o voluntário mais adequado ao paciente. . O objetivo é preparar os dados e as interfaces para a próxima etapa.

  3. Fase 3 - Integrações e testes

    Foco na integração dos módulos desenvolvidos. Esta fase conecta as diferentes estruturas ao banco de dados e ativa a troca de informações entre os módulos. Após a integração, a prioridade é testar o programa para garantir que ele funcione corretamente.

  4. Fase 4 - Expansão e Manutenção

    Implementação de novas funcionalidades, entre outras alterações baseadas no feedback da ONG, seguida por uma fase de testes destas mesmas mudanças.
